# Descrition
This is a PoC to encrypt all files in a directory with rust.

The program has a harcoded key and harcoded directory in the source code.
Feel free to change that.

# How to build the executable?
```azure
cargo build --release
```

# How to install it?
```azure
cargo install
```

# How to use it?
```azure
encrypt_dir
```
